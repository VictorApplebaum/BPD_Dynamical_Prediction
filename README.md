# BPD_Dynamical_Prediction
Code for the Dynamical BPD prediction currently in my PhD thesis but likely to be sent for publication in future. Data for this project is not currently publicly available.

Instructions:
1. Run ClinModelTraining.R with 200 or so seeds to get models ready for multiple imputation
2. Run ModelTraining.R to fit the main model
3. Run LKV3predictorBatch.R to make predictions
4. Run Evaluator.R to evaluate quality of predictions
